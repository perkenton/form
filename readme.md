Валидация полей
-------------
#### [Посмотреть страницу][2]

Проектная работа [Яндекс.Практикума][1].
Написать шаблоны регулярных выражений для валидации четырех полей:

- имя
- e-mail
- телефон
- сайт

Шаблоны записаны в атрибут ```pattern``` каждого поля.
